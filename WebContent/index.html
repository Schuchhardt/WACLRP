¿Qué es WACLRP?

Aplicación Web para el problema de localización y ruteo.

Esta aplicación está basada en una larga investigación realizada por un equipo de científicos dedicados a la resolución de problemas de optimización combinatoria, principalmente relacionados con la localización de depósitos y ruteo de vehículos. El resultado previo a esta aplicación es un programa computacional basado en un algoritmo Tabú Search Granular híbrido de dos fases descrito en el artículo científico “A two-phase hybrid heuristic algorithm for the capacitated location-routing problem” de J.W. Escobar, R. Linfati y P. Toth, publicado el año 2013 en la revista ISI Computers & Operations Research.

El objetivo de esta aplicación tiene directa relación con el extender y compartir los resultados mediante gráficos visualmente atractivos y claros para todo tipo de usuario que tenga algún interés o esté relacionado en un clásico problema de optimización. Así mismo, el poder generar nuevas instancias con el apoyo de la comunidad.

¿Qué es CLRP?

Problema de localización y ruteo con capacidad restringida.

El CLRP puede ser descrito como un problema de grafos de la siguiente manera. Sea G=(V,A) un grafo completo no dirigido, en el cual V={1, …, m + n} es el conjunto de vértices y A es el conjunto de arcos. El conjunto de vértices es dividido en dos subconjuntos. Vértices i = 1, …, m corresponden a los depósitos potenciales, cada uno con capacidad Wi y costo fijo de apertura Oi. Vértices j = m + 1, ..., m + n corresponden a los clientes, cada uno con valor de demanda Dj. Una flota homogénea de vehículos k, cada uno con capacidad Q, es disponible en cada depósito i ∈ V. Cada vehículo cuando es utilizado para realizar una ruta genera un costo fijo F. Un costo no negativo cij es asociado con cada arco (i,j) ∈ A.

El objetivo del CLRP es determinar los depósitos a abrir, los clientes a asignar a cada depósito abierto y las rutas a construir para satisfacer la demanda de los clientes con el mínimo costo global. Este costo incluye los costos fijos de apertura de los depósitos, los costos fijos asociados a los vehículos usados por las rutas y los costos de los arcos visitados por cada ruta.

Para el CLRP se imponen las siguientes reglas: Cada ruta debe comenzar y terminar en el mismo almacén; cada cliente es visitado una sola vez por un solo camino; la suma de las demandas de los clientes visitados por todas las vías no debe superar la capacidad de los vehículos; la suma de las demandas de los clientes asignados a cada uno depósito no debe superar su correspondiente capacidad; No se permiten conexiones entre depósitos.
